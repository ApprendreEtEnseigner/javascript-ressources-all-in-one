Un langage compilé

Quoi ?

Est un langage de programmation dont les implémentations sont généralement des compilateurs, c'est-à-dire des programmes qui transforment le code source en code objet.



Un langage compilé

Qui ?

Est utilisé par les développeurs qui veulent créer des programmes exécutables par une machine, avec un contrôle plus fin des aspects matériels comme la gestion de la mémoire et l'utilisation du processeur.

Un langage compilé

Où ?

Est indépendant de la machine cible, c'est-à-dire qu'il peut être exécuté sur différents types de processeurs ou de systèmes d'exploitation, à condition qu'il existe un compilateur adapté.
