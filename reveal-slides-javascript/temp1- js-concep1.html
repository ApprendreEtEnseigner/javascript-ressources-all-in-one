Un langage de programmation

Qui ?

Est utilisé par un programmeur, c'est-à-dire une personne qui...

...écrit du code source, et par un ordinateur, c'est-à-dire...

...une machine qui exécute le code source ou le code binaire obtenu après traduction.

Un langage de programmation

Peut être utilisé dans différents domaines d'application, tels que...

le développement web, le développement logiciel, le développement mobile,...

...la science des données, l'intelligence artificielle, etc.
